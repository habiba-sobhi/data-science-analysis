# E-Commerce Transactions Data Analysis

## 📌 Description
A data science project focused on analyzing customer purchasing behavior using an e-commerce transactions dataset. The project applies data cleaning, visualization, and machine learning techniques to extract insights and identify customer segments.

## 🎯 Objectives
- Analyze customer spending patterns
- Identify popular product categories
- Explore payment method preferences
- Segment customers based on purchasing behavior

## 📊 Dataset
- Source: Kaggle – E-Commerce Transactions Dataset  
- Records: 50,000 transactions  
- Features: Customer details, product category, purchase amount, payment method, transaction date  
- Type: Synthetic but realistic transactional dataset  

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

## 🔍 Data Processing
- Checked data structure and quality  
- Handled missing values and duplicates  
- Converted date column to datetime format  
- Detected and removed outliers using IQR method  

## 📈 Data Visualization
- Product category distribution (bar chart)  
- Purchase amount distribution (histogram)  
- Payment method distribution (pie chart)  
- Outlier detection using boxplots  

## 🤖 Machine Learning
- Applied **K-Means Clustering**  
- Grouped customers into segments based on:
  - Purchase Amount  
  - Age  
- Standardized data before clustering  

## 💡 Key Insights
- Some product categories have higher transaction volumes  
- Customer spending is concentrated within a specific range  
- Payment methods show clear user preferences  
- Customers can be segmented into distinct groups for targeted marketing
